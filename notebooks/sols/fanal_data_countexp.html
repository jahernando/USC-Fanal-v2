Estimate the \(\mathcal{T}_{1/2}^{\beta\beta0\nu}\) FC CL intervals#

Compute the CI for \(\mathcal{T}_{1/2}^{\beta\beta}\) at 95 % CL for:

  • \(n^{\beta\beta}_{RoI}\) events

  • \(\epsilon^{\beta_\beta}_{RoI}\), the selection eficiency, and \(a\), the fidutial acentance.

  • exposure, kg, y

def tau(nbb):
    acc_bb   = collpars.acc_bb
    eff      = collpars.eff_bb_RoI
    exposure = collpars.exposure
    tau      = fn.half_life(nbb, exposure, acc_bb * eff)
    return tau

for fc, cl in zip(fcs, cls):
    ci   = fc(n0)
    taus = [tau(x) for x in ci]
    taus.reverse()
    print('FC at {:2.0f} % CL  interval, expected bkg n = {:6.3f} , observed n = {:d}  = ({:4.2e}, {:4.2e})'.format(100*cl, nbkg, n0, *taus))

Estimate the p-value of the null hypothesis#

Consider that there is no signal, and compute the p-value, that correspond to the probability to obtain experiments with \(n >= n^{obs}_{RoI}\), if the mean is \(n^{bkg}_{RoI}\)

p0 = stats.poisson.sf(nobs, nbkg) 
print('p0-values = {:1.2e} for nobs = {:d} and nbkg = {:6.3f}'.format(p0, nobs, nbkg))

p0-values = 2.19e-02 for nobs = 36 and nbkg = 25.782

Compute the estimated \(\mathcal{T}_{1/2}^{\beta\beta}\) for \(n^{\beta\beta}_{RoI} = n^{obs}_{RoI} - n^{bkg}_{RoI}\)

nbb = nobs - nbkg if nobs > nbkg else 0
tau_countexp = tau(nbb)
