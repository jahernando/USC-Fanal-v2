# USC-Fanal

An open exercise for physics students: a neutrino-less double beta decay search using mock data inspired by the [NEXT experiment](https://next.ific.uv.es/).

## Content

Students form collaborations, analyse simulated detector data, estimate backgrounds, search for a signal, and present their results — following the workflow of a real particle physics experiment.

- **Exercise notebooks** — guided analysis from selection cuts to signal hypothesis testing.
- **Reference material** — crib sheets for pandas/plotting and fitting tools.
- **Course guides** — dynamics, deliverables, and evaluation criteria.

Entry point: `notebooks/fanal.ipynb`
